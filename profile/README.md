# Bug Labs

Bug Labs is a small development organization focused on building Minecraft plugins, tools, and other software projects.

Most of our work is centered around creating practical and polished projects, with Minecraft plugin development being one of the main areas currently. Plugins are published on Modrinth, while source code and other projects are maintained here on GitHub.

## What We Work On

- Minecraft plugins
- Developer tools and utilities
- Experimental and side projects
- Open-source software

## Platforms

- GitHub for source code and collaboration
- Modrinth for plugin releases and distribution

## Goal

The purpose of Bug Labs is to keep projects centralized, organized, and professional while continuing to experiment and build new things over time.
